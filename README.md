# shinyverse

This repository contains the initial scaffold for the "shinyverse" project.

It was initialized by an automated assistant as a local git repository inside the existing workspace `atlas-gui`.

Contents:
- README.md (this file)
- LICENSE (MIT)
- .gitignore

How to push to GitHub:
1. Create a new empty repository on GitHub named `shinyverse`.
2. Set the remote and push:

```powershell
cd shinyverse
git remote add origin https://github.com/<your-username>/shinyverse.git
git branch -M main
git push -u origin main
```
